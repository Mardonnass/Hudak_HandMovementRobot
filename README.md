# Robot Project – README

## Overview

Tento repozitár obsahuje kódy pre nášho robota, ktoré pokrývajú rôzne aspekty jeho fungovania – od základného ovládania až po pokročilejšie spracovanie dát a rozhodovanie. Projekt slúži ako experimentálne aj praktické riešenie pre vývoj autonómneho (alebo semi-autonómneho) systému.

## Funkcionalita

* Ovládanie pohybu robota
* Spracovanie vstupov zo senzorov
* Rozhodovacia logika (AI / algoritmy)
* Testovacie a experimentálne skripty

## Poznámky ku kódu

* Kód je písaný skôr experimentálne – nie všetko je finálne optimalizované.
* Niektoré časti môžu byť určené len na testovanie alebo ladenie.
* Komentáre v kóde vysvetľujú základnú logiku, ale nie vždy do hĺbky.
